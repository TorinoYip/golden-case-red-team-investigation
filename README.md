## 核心参考蓝本

- 《数据驱动的电力系统暂态稳定评估方法综述》（范士雄等）

该文献是写作与方法蓝本，不代表后续研究将复制其主题、观点或具体表述。后续工作应结合目标问题重新开展文献检索、证据核验、分类设计与原创写作。
